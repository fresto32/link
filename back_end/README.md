# README

Backend orchestrator for Link. This backend will perform CRUD operations for cards and user management.

## Usage

### Install TypeScript

```bash
npm install typescript -g
```

### Install Project Dependencies

```bash
npm install
```

### Start Development Server

```bash
npm run start-dev
```

### Serving the front end

```bash
npm run start
```
