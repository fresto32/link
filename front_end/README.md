# README

This project contains the front end for Link. Link is built exclusively with THREE js and no other graphical library.

## Usage

### Install TypeScript

```bash
npm install typescript -g
```

### Install Project Dependencies

```bash
npm install
```

### Start

```bash
npm start
```
